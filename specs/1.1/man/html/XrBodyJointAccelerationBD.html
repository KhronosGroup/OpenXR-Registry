XrBodyJointAccelerationBD(3) Manual Page

Name

XrBodyJointAccelerationBD - Body tracking joint acceleration

C Specification

The XrBodyJointAccelerationBD structure is defined as:

Member Descriptions

  • accelerationFlags is a bitfield, with bit masks defined in XrSpaceAccelerationFlagBitsBD, to indicate which members contain valid data. If none of the bits are set, no other fields in this structure should be considered to be valid or meaningful.

  • linearAcceleration is the linear acceleration of XrBodyJointLocationsBD::jointLocations with respect to and expressed in the reference frame of XrBodyJointsLocateInfoBD::baseSpace.

  • angularAcceleration is the relative angular acceleration of XrBodyJointLocationsBD::jointLocations with respect to XrBodyJointsLocateInfoBD::baseSpace.
