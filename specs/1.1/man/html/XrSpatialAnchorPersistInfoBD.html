XrSpatialAnchorPersistInfoBD(3) Manual Page

Name

XrSpatialAnchorPersistInfoBD - Info of persisting a spatial anchor

C Specification

The XrSpatialAnchorPersistInfoBD structure is defined as:

// Provided by XR_BD_spatial_anchor
typedef struct XrSpatialAnchorPersistInfoBD {
    XrStructureType            type;
    const void*                next;
    XrPersistenceLocationBD    location;
    XrAnchorBD                 anchor;
} XrSpatialAnchorPersistInfoBD;

Members

Member Descriptions

  • type is the XrStructureType of this structure.

  • next is NULL or a pointer to the next structure in a structure chain. No such structures are defined in core OpenXR or this extension.

  • location is the XrPersistenceLocationBD to persist the spatial anchor.

  • anchor is the XrAnchorBD handle of the spatial anchor.

Description

Valid Usage (Implicit)

  • The XR_BD_spatial_anchor extension must be enabled prior to using XrSpatialAnchorPersistInfoBD

  • type must be XR_TYPE_SPATIAL_ANCHOR_PERSIST_INFO_BD

  • next must be NULL or a valid pointer to the next structure in a structure chain

  • location must be a valid XrPersistenceLocationBD value

  • anchor must be a valid XrAnchorBD handle
