xrGetSpaceRoomMeshMETA(3) Manual Page

Name

xrGetSpaceRoomMeshMETA - Gets an XrRoomMeshMETA that represents a room mesh

C Specification

The xrGetSpaceRoomMeshMETA function is defined as:

// Provided by XR_META_spatial_entity_room_mesh
XrResult xrGetSpaceRoomMeshMETA(
    XrSpace                                     space,
    const XrSpaceRoomMeshGetInfoMETA*           getInfo,
    XrRoomMeshMETA*                             roomMeshOutput);

Parameters

Parameter Descriptions

  • space is a handle to an XrSpace.

  • getInfo is the input parameter that points to a valid XrSpaceRoomMeshGetInfoMETA.

  • roomMeshOutput is the output parameter that points to a valid XrRoomMeshMETA.

Description

The xrGetSpaceRoomMeshMETA function is used by the application to perform the two calls required to obtain a room mesh associated with a spatial entity specified by space.

If the spatial entity space does not have the XR_SPACE_COMPONENT_TYPE_ROOM_MESH_META component type enabled, this function must return XR_ERROR_SPACE_COMPONENT_NOT_ENABLED_FB.

Valid Usage (Implicit)

  • The XR_META_spatial_entity_room_mesh extension must be enabled prior to calling xrGetSpaceRoomMeshMETA

  • space must be a valid XrSpace handle

  • getInfo must be a pointer to a valid XrSpaceRoomMeshGetInfoMETA structure

  • roomMeshOutput must be a pointer to an XrRoomMeshMETA structure

Return Codes
On success, this command returns

  • XR_SUCCESS

  • XR_SESSION_LOSS_PENDING

On failure, this command returns

  • XR_ERROR_FEATURE_UNSUPPORTED

  • XR_ERROR_FUNCTION_UNSUPPORTED

  • XR_ERROR_HANDLE_INVALID

  • XR_ERROR_INSTANCE_LOST

  • XR_ERROR_RUNTIME_FAILURE

  • XR_ERROR_SESSION_LOST

  • XR_ERROR_SIZE_INSUFFICIENT

  • XR_ERROR_SPACE_COMPONENT_NOT_ENABLED_FB

  • XR_ERROR_VALIDATION_FAILURE
