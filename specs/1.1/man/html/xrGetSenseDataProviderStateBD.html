xrGetSenseDataProviderStateBD(3) Manual Page

Name

xrGetSenseDataProviderStateBD - Get the state of a sense data provider

C Specification

The xrGetSenseDataProviderStateBD function is defined as:

// Provided by XR_BD_spatial_sensing
XrResult xrGetSenseDataProviderStateBD(
    XrSenseDataProviderBD                       provider,
    XrSenseDataProviderStateBD*                 state);

Parameters

Parameter Descriptions

  • provider is the XrSenseDataProviderBD handle.

  • state is a pointer to the XrSenseDataProviderStateBD.

Description

The application can get a sense data provider’s state by calling xrGetSenseDataProviderStateBD.

Figure 1. XrSenseDataProviderStateBD
Valid Usage (Implicit)

  • The XR_BD_spatial_sensing extension must be enabled prior to calling xrGetSenseDataProviderStateBD

  • provider must be a valid XrSenseDataProviderBD handle

  • state must be a pointer to an XrSenseDataProviderStateBD value

Return Codes
On success, this command returns

  • XR_SUCCESS

  • XR_SESSION_LOSS_PENDING

On failure, this command returns

  • XR_ERROR_FUNCTION_UNSUPPORTED

  • XR_ERROR_VALIDATION_FAILURE

  • XR_ERROR_RUNTIME_FAILURE

  • XR_ERROR_HANDLE_INVALID

  • XR_ERROR_INSTANCE_LOST

  • XR_ERROR_SESSION_LOST

  • XR_ERROR_SPATIAL_SENSING_SERVICE_UNAVAILABLE_BD

  • XR_ERROR_FEATURE_UNSUPPORTED

See Also

XR_BD_spatial_sensing, XrSenseDataProviderBD, XrSenseDataProviderStateBD
