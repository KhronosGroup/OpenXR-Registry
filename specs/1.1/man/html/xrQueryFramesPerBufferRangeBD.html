xrQueryFramesPerBufferRangeBD(3) Manual Page

Name

xrQueryFramesPerBufferRangeBD - Query supported frames per buffer range

C Specification

The xrQueryFramesPerBufferRangeBD function is defined as:

// Provided by XR_BD_spatial_audio_rendering
XrResult xrQueryFramesPerBufferRangeBD(
    XrSession                                   session,
    XrAudioSampleRateBD                         sampleRate,
    uint32_t*                                   min,
    uint32_t*                                   max);

Parameters

Parameter Descriptions

  • session is the XrSession to query supported frames per buffer range.

  • sampleRate is the sample rate for which the supported frames per buffer range is queried.

  • min is a pointer to an uint32_t that receives the minimum supported frames per buffer.

  • max is a pointer to an uint32_t that receives the maximum supported frames per buffer.

Description

Before creating the XrSpatialAudioRendererBD, applications should call xrQueryFramesPerBufferRangeBD to query supported frames per buffer range. Applications must set XrSpatialAudioRendererCreateInfoBD::framesPerBuffer to a value within the supported range, otherwise the runtime must return XR_ERROR_VALIDATION_FAILURE from xrCreateSpatialAudioRendererBD.

The values enumerated by this function must not change during the lifetime of an XrInstance.

Valid Usage (Implicit)

  • The XR_BD_spatial_audio_rendering extension must be enabled prior to calling xrQueryFramesPerBufferRangeBD

  • session must be a valid XrSession handle

  • sampleRate must be a valid XrAudioSampleRateBD value

  • min must be a pointer to a uint32_t value

  • max must be a pointer to a uint32_t value

Return Codes
On success, this command returns

  • XR_SUCCESS

  • XR_SESSION_LOSS_PENDING

On failure, this command returns

  • XR_ERROR_FUNCTION_UNSUPPORTED

  • XR_ERROR_HANDLE_INVALID

  • XR_ERROR_INSTANCE_LOST

  • XR_ERROR_RUNTIME_FAILURE

  • XR_ERROR_SESSION_LOST

  • XR_ERROR_VALIDATION_FAILURE
