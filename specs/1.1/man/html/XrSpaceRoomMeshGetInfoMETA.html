XrSpaceRoomMeshGetInfoMETA(3) Manual Page

Name

XrSpaceRoomMeshGetInfoMETA - Input parameter to the xrGetSpaceRoomMeshMETA function

C Specification

The XrSpaceRoomMeshGetInfoMETA structure is defined as:

// Provided by XR_META_spatial_entity_room_mesh
typedef struct XrSpaceRoomMeshGetInfoMETA {
    XrStructureType               type;
    const void*                   next;
    uint32_t                      recognizedSemanticLabelCount;
    const XrSemanticLabelMETA*    recognizedSemanticLabels;
} XrSpaceRoomMeshGetInfoMETA;

Members

Member Descriptions

  • type is the XrStructureType of this structure.

  • next is NULL or a pointer to the next structure in a structure chain. No such structures are defined in core OpenXR or this extension.

  • recognizedSemanticLabelCount is an input parameter for the application to specify the number of elements in the recognizedSemanticLabels array.

  • recognizedSemanticLabels is an input parameter for the application to specify an array of XrSemanticLabelMETA. The runtime must return only the semantic labels specified in this list in XrRoomMeshFaceMETA::semanticLabel. The runtime must return semantic labels that are not specified in this list as XR_SEMANTIC_LABEL_UNKNOWN_META.

Description

Valid Usage (Implicit)

  • The XR_META_spatial_entity_room_mesh extension must be enabled prior to using XrSpaceRoomMeshGetInfoMETA

  • type must be XR_TYPE_SPACE_ROOM_MESH_GET_INFO_META

  • next must be NULL or a valid pointer to the next structure in a structure chain

  • If recognizedSemanticLabelCount is not 0, recognizedSemanticLabels must be a pointer to an array of recognizedSemanticLabelCount valid XrSemanticLabelMETA values
